# System Architecture

## 核心模組

- Frontend（Web / Flutter）
- Admin Backend
- API Gateway
- BFF Layer
- Microservices
- Communication Service
- AI Services
- CI/CD System
- Kubernetes Infrastructure

## 核心原則

- 微服務可獨立部署
- 單元測試作為快速驗證
- E2E 作為功能完成驗證
- GitHub Actions 為開發觸發層
- Tekton 為 CI/CD 執行層
- Service Registry 支援動態更新
