# Deployment Management

版本：v0.5.1

## 模組目標
管理版本發版、環境狀態、灰度策略、回滾與審批流程。

## 核心功能
1. 環境部署狀態檢視
2. 版本列表與變更內容
3. 發版、灰度、終止灰度
4. 回滾與回滾審批
5. 部署紀錄與成功率統計

## 環境
1. `dev`
2. `test`
3. `pp`
4. `prod`

## 規則
1. 進入 `prod` 前需具備可部署標記
2. 回滾需選定穩定版本並留下原因
3. 灰度發布需有比例、觀察期間與告警門檻
4. 部署失敗需建立 incident 或 bug ticket

## 主要欄位
1. Release ID、版本號、部署環境
2. 發版人、審批人、部署時間
3. 策略、結果、回滾資訊
4. 健康檢查、告警摘要、影響服務

## 指標建議
1. 部署成功率
2. 回滾率
3. 變更失敗率
4. 平均部署時間
