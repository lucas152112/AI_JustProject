# Microservice Management

版本：v0.5.1

## 模組目標
管理服務清單、相依關係、健康度、版本與運行操作。

## 核心功能
1. 服務註冊與下線
2. 服務詳情、健康檢查、資源配置
3. 查看依賴拓樸與通訊關係
4. 擴縮容、重啟、配置變更
5. 服務版本與部署歷史查詢

## 主要欄位
1. Service ID、名稱、領域、擁有者
2. 狀態：`Healthy`、`Warning`、`Critical`、`Offline`
3. CPU/Memory、replica、版本、最後部署時間
4. 入口位址、相依服務、告警狀態

## 關鍵規則
1. 手動重啟與擴縮容需保留操作原因
2. 服務下線前需檢查依賴影響
3. 健康檢查異常需同步通知 deployment 與 reporting 模組
4. 服務配置變更需可追溯到版本與人員

## 指標建議
1. 服務健康率
2. 異常服務數
3. 平均恢復時間
4. 配置變更次數
