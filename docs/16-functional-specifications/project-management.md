# Project Management

版本：v0.5.1

## 模組目標
管理專案生命週期、成員、里程碑、版本與風險追蹤。

## 核心功能
1. 建立與封存專案
2. 維護專案成員、角色與權責
3. 管理模組、里程碑、版本與變更紀錄
4. 查看活動流、風險與阻塞事項
5. 匯出專案清單與進度摘要

## 主要欄位
1. 專案代碼、名稱、描述、負責人
2. 狀態：`Draft`、`Active`、`UAT`、`Released`、`Closed`
3. 起訖日、優先級、目標版本、標籤
4. 里程碑、版本、成員統計、風險等級

## 關鍵規則
1. 專案需至少有一位 PM 或負責人
2. 專案封存後僅允許讀取，不可再建立新任務
3. 版本必須關聯里程碑或發版批次
4. 重大範圍變更需觸發通知與重排工期

## 子功能
1. 成員管理
2. 里程碑管理
3. 版本管理
4. 專案活動流

## 指標建議
1. 專案準時率
2. 里程碑完成率
3. 活躍專案數
4. 延期專案數
