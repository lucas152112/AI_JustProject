# System Settings

版本：v0.5.1

## 模組目標
統一管理平台層級的 API Key、模型路由、通知模板與系統參數。

## 核心功能
1. API Keys 建立、輪替、停用
2. 模型設定、provider 路由與 fallback
3. 通知模板、頻率與預設渠道
4. 系統參數管理與預設值恢復
5. 變更歷史與審計查詢

## 關鍵規則
1. API Key 完整值僅首次顯示
2. 模型路由變更需可回滾
3. 系統參數修改需保留前後值
4. 高風險設定需限制 Super Admin / System Admin

## 參數類型
1. 安全參數
2. AI 模型參數
3. 通知參數
4. 平台運維參數

## 指標建議
1. 金鑰輪替頻率
2. 設定異動次數
3. 失敗配置發布次數
4. 敏感設定存取次數
