# User Management

版本：v0.5.1

## 模組目標
管理帳號生命週期、登入安全、角色綁定與使用者審計紀錄。

## 核心功能
1. 建立、編輯、停用、重新啟用使用者
2. 綁定系統角色與專案角色
3. 重設密碼、MFA 管理、裝置解除
4. 查詢登入紀錄與異常登入事件
5. 批次匯入與批次狀態調整

## 主要欄位
1. 帳號、姓名、Email、部門、職稱
2. 狀態：`Invited`、`Active`、`Suspended`、`Disabled`
3. 安全欄位：MFA 狀態、最後登入時間、最後密碼更新日
4. 角色欄位：系統角色、專案角色、群組標籤

## 狀態流轉
1. `Invited` -> `Active`
2. `Active` -> `Suspended`
3. `Suspended` -> `Active`
4. `Active/Suspended` -> `Disabled`

## 關鍵規則
1. 停用帳號後不可登入，但歷史資料保留
2. 角色變更需寫入審計日誌
3. 重設密碼與解除 MFA 屬高風險操作
4. 管理者不可停用最後一位 Super Admin

## 通知節點
1. 帳號建立邀請
2. 密碼重設通知
3. 權限變更通知
4. 異常登入告警

## 指標建議
1. 啟用帳號數
2. 停用帳號數
3. MFA 啟用率
4. 異常登入次數
